# Ignite Realtime Chat

Ignite is a feature-rich, high-performance real-time messaging application built with Next.js and Firebase. Designed for instant communication, it leverages Firebase's real-time capabilities to provide a seamless chat experience across a global user base.

## 🔥 Core Features

- **Real-time Messaging**: Instant message delivery and synchronization using Firestore.
- **Global Chatroom**: Interact with users worldwide in a unified, professional chat interface.
- **Secure Authentication**: Frictionless login via Google Account integration powered by Firebase Authentication.
- **Message History**: Persistent access to chat history, ensuring you never miss a conversation.
- **Modern Interface**: A sleek, mobile-ready UI built with Tailwind CSS, supporting deep customization.
- **Serverless Architecture**: Scalable and cost-effective deployment leveraging Vercel and Firebase.

## 🚀 Technology Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Backend Services**: [Firebase](https://firebase.google.com/) (Realtime Database, Auth)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📦 Installation & Configuration

1. **Clone the Repository**
   ```bash
   git clone https://github.com/agentic-saim09/ignite-realtime-chat
   cd ignite-realtime-chat
   ```

2. **Install Project Dependencies**
   ```bash
   npm install
   ```

3. **Set Up Environment Variables**
   Create a `.env.local` file in the root directory and add your Firebase credentials:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=your_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=your_auth_domain
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=your_project_id
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=your_storage_bucket
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
   NEXT_PUBLIC_FIREBASE_APP_ID=your_app_id
   ```

4. **Start Development Server**
   ```bash
   npm run dev
   ```
   The application will be live at [http://localhost:3000](http://localhost:3000).
